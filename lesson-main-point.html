Main Point Questions

ContextSupportThe claim being proven

Animated script

A main point answer should feel like the author's thesis, not a detail the author merely mentioned. Before looking down, say the conclusion in your own words.

Key takeaways

  • Reject answers that are true but too narrow.
  • Reject answers that state evidence instead of the conclusion.
  • Prefer the answer that captures the author's overall commitment.

Common trap

Some answers borrow exact topic words but describe the wrong sentence job.
